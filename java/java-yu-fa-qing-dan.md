# Java清单

* 主方法入口：main
* if-else
* for循环：两种
* do-while
* while
* switch-case
* 数组：一维、二维、多维
* 标识符
* 修饰符
  * 访问控制
  * 非访问控制
* 变量
  * 局部变量
  * 静态变量
  * 非静态变量
* 枚举
* 注释
* 基本数据类型
  * 8种基本类型
    * 各对应一个包装类
    * byte：8位，有符号，-2^7～2^7-1。
    * short：
    * int：
    * 6种数字类型
    * 1种字符类型
    * 1种布尔型 
* 面向对象
  * 类：对事物、逻辑、算法、概念的抽象，描述类的行为状态
  * 对象（实例）：类的实例
  * 引用
  * 构造方法
  * this
  * 方法重载：同名不同参，与返回无关。
  * 抽象类
    * 提供通用代码
    * 提供通用方法定义
    * 不能创建对象
    * 不一定有抽象方法，反之若有抽象方法，类必为抽象。
  * final
  * static
    * 静态变量（类变量）
    * 最优先加载到内存
  * 接口
    * 解耦合，多实现
    * 只能定义公共
      * 常量
      * 抽象方法
      * 内部类
      * 内部接口
  * 内部类
    * 定义在类内部、方法内部、局部代码块内部的类
    * 非静态内部类
      * 依赖外部类对象存在
      * 不能用static
    * 静态内部类
      * static修饰
      * 只能访问外部类的静态成员，不能访问非静态的
    * 局部内部类
    * 成员内部类
    * 匿名内部类
  * 三大特性
    * 继承
      * extends
      * 代码重用、复用
      * 单继承
      * 不能继承的
        * 构造方法
        * 无访问权限的成员
        * 静态成员
    * 多态
      * 编译\(声明变量\)和运行\(实例化后\)表现为不同的形态（数据类型）
      * 条件
        * 存在继承
        * 子类需重写父类方法
        * 父类引用变量指向子类对象
      * 向上转型
      * 向下转型
    * 封装
      * 访问控制符
        * private：类可见
        * default/不写：包可见
        * protected：包可见，不同包子类可见
        * public：项目可见
      * 好处
        * 隐藏实现细节
        * 只能通过规定方式访问数据
        * 方便加入存取控制代码，限制不合理操作
  * 源文件
    * 只能有一个public类
    * 可有多个非public类
